Tutorial install:
1. Install MongoDB from https://www.mongodb.com/download-center#community
2. If on Windows, follow tutorial https://docs.mongodb.com/manual/tutorial/install-mongodb-on-windows/
3. Start MongoDB database
"C:\Program Files\MongoDB\Server\4.0\bin\mongod.exe" --dbpath="[[ path to database directory ]]"
C:\Program Files\... -> default MongoDB path
3. Connect to MongoDB
"C:\Program Files\MongoDB\Server\4.0\bin\mongo.exe"
